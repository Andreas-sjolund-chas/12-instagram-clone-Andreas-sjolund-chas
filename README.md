### Deployment

```
git checkout heroku-deploy
git add <files>
git commit -m "<commit message>"
git push heroku master
```

###How to start app

```
cd client
yarn start
cd ../server
nodemon server.js
-- with debugger in server --
nodemon --inspect server.js
```
